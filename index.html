About

Project INF is part of the collaborative research center SFB/Transregio 161 with focus on Quantitative Methods for Visual Computing. As part of the project, equipment can be lent free of charge to SFB/Transregio 161 project members at the University of Konstanz. For more information, please see the How to borrow section or use the mail icon at the menu bottom, if you have any questions.

How to borrow

To borrow a device, please write an e-mail with the following information:

  1. SFB/Transregio 161's project name
  2. Responsible person's name
  3. Information about planned usage (1-2 sentences)
  4. Expected usage duration
Then to pick up or return a device, please come to PZ 1005. Upon pick-up you will be asked to sign a short loan agreement.
